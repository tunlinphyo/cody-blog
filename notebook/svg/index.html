Scalable Vector Graphics

What is Scalable Vector Graphics?

Scalable Vector Graphics, known as SVG, is an XML-based image format used to create sharp, resolution-independent graphics on the web.

SVG stays crisp at any size, supports CSS styling and animations, and is perfect for icons, logos, illustrations, and charts.

<svg>

The root element of an SVG graphic. It defines the drawing area and contains all shapes, paths, text, and groups.

viewBox

The viewBox defines the internal coordinate system of the SVG. It controls how the graphic scales inside its container.

<rect>

Draws rectangles and squares. You can control its position, width, height, corner radius, fill, and stroke.

<line>

Draws a straight line between two points. It becomes visible when a stroke is applied.

<circle>

Draws a circle using its center position and radius.

<ellipse>

Draws an oval shape. Unlike a circle, it has separate horizontal and vertical radii.

<polyline>

Draws a connected series of lines. It does not automatically close the shape.

<polygon>

Draws a closed shape made from multiple connected points.

<path>

The most powerful SVG shape. It can draw lines, curves, arcs, and complex custom illustrations using path commands.

This is a great editor for working with SVG paths.

<g>

Groups multiple SVG elements together so they can share styles, transforms, or animations.

<defs>

Stores reusable definitions such as gradients, filters, masks, and symbols without displaying them directly.

<clipPath>

A clipPath hides parts of an element outside the clipping shape.

<mask>

A mask controls transparency. White reveals, black hides, and gray creates partial transparency.

Transform

Moves, rotates, scales, or skews SVG elements.

Gradient

SVG gradients create smooth color transitions and can be reused by referencing their id.

Let's create a sample SVG menu icon using <rect> and <line>.

First, create an <svg> element with a viewBox of 0 0 24 24. This means the drawing area starts at x=0, y=0 and has a width and height of 24 units.

Add a <line> element from x1="5" y1="12" to x2="19" y2="12", then set its stroke color and stroke-width.

And then add two more <line>s width difference y units.

Add a <rect> element for the border box with x="1", y="1", and a width and height of 22 units.

Finally, group the duplicated attributes using <g>, then add stroke-linecap="round" and rx="4" to the <rect> for a rounded look.

Gradient

Put reusable gradients in <defs>, give each one a unique id, and reference it with url(#id). For strokes on lines, gradientUnits="userSpaceOnUse" keeps the gradient coordinates aligned with the SVG viewBox.

Use <radialGradient> when the color should spread outward from a center point. The cx, cy, and r attributes control the outer circle of the gradient, while fx, fy, and fr can move or resize the starting point.

Mask

Creating masks in SVG can be a little tricky. Start by drawing the mask shapes outside the <mask> so you can preview them, then move those shapes into the mask once they look right. A simple way to think about masks is that white keeps the area visible, while black cuts it out.

SVG gives you a small set of elements that can scale from simple icons to detailed illustrations. To go deeper into each element and its attributes, explore the MDN SVG element reference.
