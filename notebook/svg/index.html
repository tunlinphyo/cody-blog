Scalable Vector Graphics

What is Scalable Vector Graphics?

Scalable Vector Graphics, known as SVG, is an XML-based image format used to create sharp, resolution-independent graphics on the web.

SVG stays crisp at any size, supports CSS styling and animations, and is perfect for icons, logos, illustrations, and charts.

<svg>

The root element of an SVG graphic. It defines the drawing area and contains all shapes, paths, text, and groups.

viewBox

The viewBox defines the internal coordinate system of the SVG. It controls how the graphic scales inside its container.

<rect>

Draws rectangles and squares. You can control its position, width, height, corner radius, fill, and stroke.

<line>

Draws a straight line between two points. It becomes visible when a stroke is applied.

<circle>

Draws a circle using its center position and radius.

<ellipse>

Draws an oval shape. Unlike a circle, it has separate horizontal and vertical radii.

<polyline>

Draws a connected series of lines. It does not automatically close the shape.

<polygon>

Draws a closed shape made from multiple connected points.

<path>

The most powerful SVG shape. It can draw lines, curves, arcs, and complex custom illustrations using path commands.

This is a great editor for working with SVG paths .

<g>

Groups multiple SVG elements together so they can share styles, transforms, or animations.

<defs>

Stores reusable definitions such as gradients, filters, masks, and symbols without displaying them directly.

Attributes

SVG elements use attributes to control their rendering. Here's a list of the most commonly used attributes.

The viewBox defines the internal coordinate system of the SVG. It controls how the graphic scales inside its container.

The fill attribute sets a shape's paint color and gradients.

The stroke attribute sets the outline color of a shape, including gradients and patterns.

The stroke-width attribute sets the width of a shape or text outline and can be inherited from elements like .

The stroke-dasharray attribute defines the dash and gap pattern of a shape's outline.

The stroke-linecap attribute defines the shape of the ends of stroked lines.

Gradient

Put reusable gradients in <defs>, give each one a unique id, and reference it with url(#id). gradientUnits="userSpaceOnUse" keeps the gradient coordinates aligned with the SVG viewBox.

Mask

Creating masks in SVG can be a little tricky. Start by drawing the mask shapes outside the <mask> so you can preview them, then move those shapes into the mask once they look right.

Masking

Let's design a reset icon whit the help of <mask>.

Draw a <circle> centered at (12, 12) with a radius of 10.

Cover the entire viewBox with a white rectangle (x=0, y=0, width=24, height=24) using <rect>.

Use <polygon> to set the area to cut in coordinates like (x y, x y, ...), and fill with black.

Wrap the <rect> and <polygon> in a <mask>, then apply it to the <circle>.

Create a triangle with <polygon> using three coordinate pairs, then rotate it around the left center.

Finally, add another <polygon> to finish the reload icon.
