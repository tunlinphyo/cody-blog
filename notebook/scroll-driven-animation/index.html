Scroll-driven Animations

What are Scroll-driven Animations?

Scroll-driven animations animate elements based on scroll position instead of the default time-based timeline.

Scroll() Timeline

The scroll() timeline tracks the scroll progress of a scroll container or the page itself.

View() Timeline

The view() timeline links an animation to an element's visibility inside the scrollport.

Animation-range

animation-range controls when a scroll-driven animation starts and ends. This lets you animate only during a specific part of the element's visibility.

This feature is not Baseline yet, so please use a supported browser to test the demo in this article.
